# 달걀 아저씨 살리기

과학 역학적 에너지 단원을 활용하여 만든 과학게임

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **과학 퀴즈** | 제일 처음에 시작하기를 누르면 나오는 과학퀴즈. 총 5개가 출제되고 맞히면 예산을 얻을 수 있음. |
| **아이템 고르기** | 한정된 예산에서 아이템을 살 수 있는 기능. 달걀 아저씨를 구하는 아이템을 사야한다. |
| **수평 발사 속도** | 달걀 아저씨를 얼마나 밀어서 낙하시킬지 결정하는것. |
| **낙하 시작 위치** | 달걀 아저씨를 어디에서 낙하시킬지 결정하는것. 낙하 지점에 착지하기 위해 필요함. |
